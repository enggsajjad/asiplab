
License Server Setup Guide
--------------------------

ASIP Solutions, Inc.


1. Intall Java Runtime Environment (JRE)

When using RPM version of JRE installer, default installation
location is /usr/java/default

2. Unpack the License Server Package and Copy to any location

Recommended location is /usr/local/ASIPSLicServ

3. Copy your license file (Optional)

You may place your license file to any location

4. Edit ASIPSLicServ script to match your configuraion

Specify your Java Runtime location, the Server software location,
your license file location and name, and the TCP port number to
accept accesses from other machines.

5. Copy ASIPSLicServ script to /etc/init.d and Create symlink

Copy ASIPSLicServ to /etc/init.d
Create symbolic link at /etc/rc.d/rc3.d and rc5.d as its name
of S99ASIPSLicServ

 # cd /etc/rc.d/rc3.d
 # ln -s ../init.d/ASIPSLicServ S99ASIPSLicServ

6. Manually start server (Optional)

License server starts every time when the machine boots.
You may start the license server manually by service command.

 # service ASIPSLicServ start

7. Trouble Shooting

Log files are saved in the License Server installation directory.

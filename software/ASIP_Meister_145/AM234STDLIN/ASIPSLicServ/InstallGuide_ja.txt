
ライセンス サーバー セットアップ ガイド
=======================================

エイシップ・ソリューションズ株式会社


1. Javaランタイム環境(JRE)のインストール

OracleのサイトからJava Runtime Environmentをダウンロード
してインストールします．

RPM版のインストーラを使用した場合，/usr/java/defaultに
インストールされます．

2. ライセンス サーバー パッケージを展開し，任意の位置にコピー

推奨位置は /usr/local/ASIPSLicServ です．

3. ライセンス ファイルのコピー (オプション)

ライセンス ファイルは任意の位置に置くことができます．

4. ASIPSLicServスクリプトの編集

Javaランタイム，ライセンス サーバー ソフトウェア，および
ライセンス ファイルの位置をお使いの環境に合わせて編集します．

また，サーバーが通信に使用するTCPポートの番号も指定します．

5. ASIPSLicServスクリプトを/etc/init.dにコピーし，シンボリック リンクを作成

ASIPSLicServスクリプトを/etc/init.dディレクトにコピーします．

コピーしたファイルへのシンボリック リンクを/etc/rc.d/rc3.d および
/etc/rc.d/rc5.d に S99ASIPSLicServ の名前で作成します．

 # cd /etc/rc.d/rc3.d
 # ln -s ../init.d/ASIPSLicServ S99ASIPSLicServ

6. 手動でのライセンス サーバーの起動 (オプション)

ライセンス サーバーはマシン起動時に自動的に開始されます．
手動でライセンス サーバーを起動するには，以下のコマンドを
実行します．

 # service ASIPSLicServ start

7. トラブル シューティング

ログファイルはライセンス サーバーのインストール先に作成されます．
